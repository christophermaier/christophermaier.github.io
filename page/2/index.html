Blogging on Github with Jekyll

As per the instructions on the Jekyll install page, gem install jekyll then (since I didn't have this set already): export PATH=${PATH}:/Users/maier/.gem/ruby/1.8/bin added to my Bash ~/.profile file. Pygments is cool for syntax highlighting. On a Mac with MacPorts, it's as easy as this: sudo port install python25 py25-pygments Running Jekyll with its standalone server is great for testing your site locally: jekyll --auto --server That'll run an embedded web server at http://localhost:4000 (by default); anytime you change any of your site files, Jekyll will reprocess them and make them available immediately.
